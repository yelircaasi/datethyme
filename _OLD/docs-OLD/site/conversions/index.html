Span Creation

Assume the following variable types in the following examples:

d, d0, d1 = Date(...), Date(...), Date(...)
t, t0, t1: Time
ds: DateSpan
ts: TimeSpan
dts: DateTimeSpan

A span instance can be created from:

  • two point types: (d0 >> d1) == DateSpan(d0, d1)
  • a point type and a span type:
    • (d >> ds) == DateSpan(d, ds.end)
    • (t >> ts) == TimeSpan(t, ts.end)
  • a span type and a point type:
import tensorflow as tf

(d0 >> d1) == DateSpan(d0, d1)

The range() function is used to generate a sequence of numbers.
